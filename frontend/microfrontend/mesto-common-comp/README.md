# mesto-common-comp

> Common Components and Controls for Mesto project

## Install

```bash
npm install --save mesto-common-comp
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'mesto-common-comp'
import 'mesto-common-comp/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
